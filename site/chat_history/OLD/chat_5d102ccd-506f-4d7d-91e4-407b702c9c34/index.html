    He creado el directorio de scripts en tu espacio de trabajo conteniendo los archivos batch (.bat) solicitados para simplificar todo tu flujo de trabajo desde la consola CMD de Windows.

    Los archivos creados se encuentran en la carpeta scripts/:

    1. 📥 scripts/install.bat
    2. Crea el entorno virtual .venv automáticamente (si no existe).
    3. Actualiza pip.

    4. Instala de manera limpia todas las dependencias listadas en el requirements.txt.

    5. 🚀 scripts/run.bat

    6. Activa el entorno virtual de Python.

    7. Arranca el servidor FastAPI con Uvicorn en el puerto 8000 con recarga automática activada.

    8. 🛠️ scripts/setup_aliases.bat

    9. Registra de forma instantánea dos macros/alias de Windows (doskey) para la sesión actual del CMD.
    10. Permite invocar las tareas simplemente escribiendo en la terminal:
      • install (para configurar el ambiente).
      • run (para iniciar el servidor).

    Instrucciones de Uso

    1. Abre tu terminal de Windows (CMD).
    2. Navega al directorio del proyecto o directamente ejecuta el configurador de alias:
    scripts\setup_aliases.bat
    1. A partir de ese momento, en esa misma ventana de CMD, puedes ejecutar las siguientes palabras clave abreviadas:

    2. Para reinstalar/configurar dependencias:

      install

    3. Para encender el servidor y empezar a probar el API y el dashboard interactivo:

      run

    Elementos Ignorados

    1. Entornos Virtuales: Carpetas habituales de Python como .venv/, venv/, env/ y logs de pip.
    2. Caché y Compilados: Archivos temporales de ejecución como __pycache__/, *.pyc, *.pyo y cachés de análisis estático (.pytest_cache/, .mypy_cache/, etc.).
    3. Editores e IDEs: Configuraciones de entornos populares como VS Code (.vscode/) y JetBrains PyCharm/WebStorm (.idea/), además de archivos temporales de editores de texto (*.swp, *~).
    4. Sistema Operativo: Archivos temporales generados por Windows y macOS (Thumbs.db, .DS_Store).
    5. Logs: Archivos de registro de errores y eventos con extensión *.log.

    Quedo a tu disposición si necesitas agregar alguna exclusión específica adicional.
